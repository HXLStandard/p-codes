Experimental p-coding site

This site contains GeoJSON files with names matching humanitarian p-codes (place codes).

Usage

The files use the following naming scheme:

http://hxlstandard.github.io/p-codes/{COUNTRY}/{P-CODE}/{TYPE}.json

Parameters

{COUNTRY}
An ISO 3166-1 alpha3 code for the country, e.g. "MLI" for Mali.
{P-CODE}
The humanitarian p-code, e.g. "MLI001002" for the Bafoulabé Cercle (ADM2) of the Kayes Region (ADM1) of Mali.
{TYPE}
Currently only "bounds" is present (for the boundary [multi]polygon), but the site will eventually support other feature types.

Example

Download the GeoJSON polygon for the bounds of the Bafoulabé Cercle of the Kayes Region of Mali (p-code "MLI001002"):

http://hxlstandard.github.io/p-codes/MLI/MLI001002/bounds.json

Current coverage

MLI
Mali (ADM0, ADM1, ADM2, ADM3)
SOM
