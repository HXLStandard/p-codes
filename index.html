Experimental p-coding site

This site contains GeoJSON files with names matching UNHCR p-codes (place codes).

Usage

The files use the following naming scheme:

https://unhcr-mena.github.io/p-codes/{COUNTRY}/{admin-level}.geojson

Parameters

{COUNTRY}
An ISO 3166-1 alpha3 code for the country, e.g. "JOR" for Jordan.
{admin-level}
The requested admin-level (ADM1, ADM2, ADM3, ADM4) for the country.
{TYPE}
geoJson present boundaries using [multi]polygon.

Maintain

The files are text based and can be maintained directly in Github. please send a request to get access this repository

Example

Download the GeoJSON polygon for admin level 1 of Jordan (Governorates):

http://unhcr-mena.github.io/p-codes/JOR/ADM1.geojson
