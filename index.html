Experimental p-coding site

This site contains GeoJSON files with names matching UNHCR p-codes (place codes) for administrative units. All geographic files are formatted in GeoJson managed under version control. This site is based on an idea from the Open Knowledge Institute and is forked from a test from the Humanitarian Exchange Language (HXL) team

Parameters

{COUNTRY}
An ISO 3166-1 alpha3 code for the country, e.g. "JOR" for Jordan.
{admin-level}
The requested admin-level (ADM1, ADM2, ADM3, ADM4) for the country.
{TYPE}
geoJson present boundaries using [multi]polygon.

Example

Download the GeoJSON polygon for admin level 1 of Jordan (Governorates):

https://github.com/unhcr-mena/p-codes/blob/gh-pages/JOR/ADM1.geojson

Usage

The files use the following naming scheme:

https://unhcr-mena.github.io/p-codes/{COUNTRY}/{admin-level}.geojson

The geoJSON map can also be placed in any HTML page that supports javascript (e.g., GitHub Pages) and embedded within an Iframe.

For example, for Jordan, the URL to the raw file is https://raw.githubusercontent.com/unhcr-mena/p-codes/gh-pages/JOR/ADM1.geojson , the embed code would be:

<iframe src="https://render.githubusercontent.com/view/geojson?url=https://raw.githubusercontent.com/unhcr-mena/p-codes/gh-pages/JOR/ADM1.geojson" width="620" height="420" frameborder="0"> </iframe>

By default, the embedded map size is 420px x 620px, but you can customize the output by passing height and width variables as parameters at the end, such as height=300&width=500.

it also possible to read this datset directly from Qgis.

Maintain

The files are text based and can be maintained directly in Github. please send a request to get access this repository

An online editor like geojson.io can be sued to edit the file.
